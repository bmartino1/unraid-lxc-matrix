==================================================
WHY DEBIAN BOOKWORM (12) AND NOT BUSTER (10)?
==================================================
Buster (10) reached EOL August 2022.
Bookworm (12) ships OpenJDK 17 natively (required for JVB2/jicofo).
Jitsi's stable APT repo supports bookworm.
Matrix Synapse packages support bookworm.
PostgreSQL 16 APT repo supports bookworm.
=> Bookworm is the correct base for this stack.

==================================================
NETWORK ARCHITECTURE
==================================================
LXC container acts as the entire stack.
All traffic enters via Nginx on port 80/443.

Port 80:  HTTP -> HTTPS redirect + ACME challenge
Port 443: Nginx stream with ssl_preread SNI routing:
           <domain>         -> Element Web (Nginx http, port 8443)
           matrix.<domain>  -> Matrix Synapse (Nginx http -> :8008)
           meet.<domain>    -> Jitsi Meet (Nginx http -> prosody/JVB)

Port 3478: coturn TURN/STUN (UDP+TCP)
Port 5349: coturn TURNS (TLS)
Port 10000/UDP: Jitsi Video Bridge media

Internal only:
  8008  Matrix Synapse HTTP
  8443  Nginx internal HTTPS (SNI-terminated)
  5280  prosody BOSH/XMPP
  5432  PostgreSQL
  6379  Valkey
  9090  JVB Colibri HTTP
